Semana 3:

Herramienta de texto: Manipular texto, texto artístico Adaptar texto a trayecto





Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta seleccion Herramienta forma Herramienta linea Herramienta círculo Herramienta Texto

PROCEDIMIENTO

    Manipulación de Texto Artístico

  1. Seleccionar la Herramienta de Texto

    Abre CorelDRAW y selecciona la herramienta de texto (tecla F8).

  2. Crear Texto Artístico

    Haz clic en el área de trabajo y escribe el texto. Modifica fuente, tamaño, color y aplica efectos desde la barra de propiedades.

  3. Aplicar Estilos

    Con el texto seleccionado, utiliza la barra de propiedades para aplicar negrita, cursiva, subrayado y cambiar la alineación.

    Adaptar Texto a un Trayecto

  1. Dibujar el Trayecto

    Usa la herramienta de forma para crear el trayecto (línea, curva o forma).

  2. Seleccionar el Texto

    Asegúrate de que el texto esté seleccionado.

  3. Adaptar Texto al Trayecto

    Con el texto y el trayecto seleccionados, ve a Texto en el menú y selecciona Adaptar texto a trayecto.

  4. Ajustes Finos

    Ajusta la posición del texto a lo largo del trayecto arrastrándolo o utilizando las opciones de alineación.

Semana 4:

Soldar, recortar, intersección y mezcla interactiva
Herramientas: soldar, recortar intersección, delante, atrás, mezcla interactiva, efecto de transparencia




Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta seleccion Herramienta forma Herramienta linea Herramienta círculo Herramienta Mezcla Interactiva

PROCEDIMIENTO

    1. Soldar

  1. Seleccionar Objetos: Crea o selecciona los objetos que deseas soldar. Asegúrate de que se superpongan.
  2. Usar la Herramienta Soldar: Ve a la barra de herramientas y selecciona la herramienta de soldar (puedes encontrarla en el menú Objeto > Combinar > Soldar).
  3. Aplicar Soldadura: Haz clic en los objetos que deseas combinar. Los objetos se unirán en un solo objeto.

    2. Recortar

  1. Seleccionar Objetos: Crea un objeto que servirá como máscara (por ejemplo, un rectángulo o una forma) y colócalo sobre el objeto que deseas recortar.
  2. Usar la Herramienta Recortar: Selecciona ambos objetos (el objeto a recortar y la máscara), luego ve a Objeto > Recortar > Recortar.
  3. Resultado: El objeto original se recortará según la forma de la máscara.

    3. Intersección

  1. Seleccionar Objetos: Crea o selecciona los objetos que deseas intersectar.
  2. Usar la Herramienta Intersección: Selecciona ambos objetos y ve a Objeto > Intersección.
  3. Aplicar Intersección: Solo la parte donde los objetos se superponen se mantendrá, creando un nuevo objeto.

    4. Mezcla Interactiva

  1. Crear Objetos Base: Dibuja dos o más objetos que desees mezclar (por ejemplo, círculos o rectángulos).
  2. Seleccionar Objetos: Selecciona los objetos que deseas mezclar.
  3. Usar la Herramienta Mezcla Interactiva: Ve a Efectos > Mezcla Interactiva y selecciona Crear Mezcla.
  4. Ajustar la Mezcla: Puedes ajustar la cantidad de pasos en la mezcla y el espaciado entre ellos en la barra de propiedades.

    5. Efecto de Transparencia

  1. Seleccionar Objeto: Selecciona el objeto al que deseas aplicar el efecto de transparencia.
  2. Aplicar Efecto de Transparencia: Ve a Efectos > Transparencia. Se abrirá una ventana de propiedades.
  3. Ajustar Transparencia: Utiliza el control deslizante para ajustar el nivel de transparencia según lo desees. También puedes elegir el tipo de transparencia (uniforme, degradado, etc.).

    6. Ordenar Objetos

  1. Traer al Frente o Enviar al Fondo: Selecciona el objeto, ve a Objeto > Ordenar y elige Traer al frente o Enviar al fondo según sea necesario.

Semana 5:

Herramienta de dibujo libremente a mano alzada: Dibujo a mano alzada Dibujo inteligente


Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta seleccion Herramienta forma Herramienta linea Herramienta B-Spline Herramienta Mano Alzada

PROCEDIMIENTO

  1. Selecciona la Herramienta de Dibujo a Mano Alzada y dibuja el contorno básico del logo.
  2. Activa el Dibujo Inteligente para corregir y mejorar la precisión de las líneas.
  3. Ajusta el dibujo utilizando la herramienta de selección para mover, escalar o rotar las partes según sea necesario.
  4. Agrega detalles adicionales, como sombras, contornos o elementos decorativos, usando la herramienta de lápiz o forma.
  5. Modifica los nodos del dibujo a mano alzada para ajustar la forma del logo según sea necesario.
  6. Aplica color utilizando rellenos sólidos, degradados o patrones.
  7. Agrega efectos de estilo, como sombras, transparencias o contornos, para darle un aspecto profesional al logo.
  8. Guarda el proyecto en formato .cdr y exporta el logo en el formato deseado (PNG, JPEG, SVG, etc.).
  9. Revisa el diseño final para asegurarte de que todos los elementos estén en su lugar y el diseño sea coherente.

Semana 6:

Herramienta de dibujo a partir de nodos:Dibujar a partir de nodos Herramientas Bézier Convertir en curvas.




Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta Relleno Herramienta Forma Herramienta Linea Herramientas Bézier Herramienta Pluma

PROCEDIMIENTO

  1. Selecciona la Herramienta Bézier de la barra de herramientas para crear líneas y curvas precisas a partir de nodos.
  2. Dibuja el contorno general del logo, haciendo clic para crear nodos y arrastrando para crear curvas suaves. Sigue la forma básica del pájaro, incluyendo la cabeza y el pico.
  3. Si es necesario, convierte el objeto a curvas para poder editar los nodos y las curvas con más precisión.
  4. Usa la herramienta de selección de nodos para ajustar la posición y forma de los nodos, asegurándote de que las curvas sean suaves y la forma general sea precisa.
  5. Agrega detalles como los ojos, el pico y las cejas utilizando la herramienta Bézier. Mantén la proporción y estilo del logo original.
  6. Selecciona las diferentes partes del logo y aplica los colores correspondientes utilizando la paleta de colores. Usa rellenos sólidos vibrantes para las áreas principales.
  7. Si lo deseas, agrega efectos de estilo como sombras o contornos para darle un aspecto más profesional al logo.
  8. Guarda el proyecto en formato .cdr y exporta el logo en el formato deseado (PNG, JPEG, SVG, etc.).
  9. Haz una revisión final para asegurarte de que todos los elementos estén en su lugar y el diseño sea coherente.

Semana 7:

Extrucción, envoltura y silueta: Efectos tridimensionales Extrusión, Envoltura Silueta interactiva



Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta seleccion Herramienta Texto Herramienta Silueta Herramienta Envoltura Herramienta Extruccion

PROCEDIMIENTO

  1. Crea el diseño base del gatito utilizando la herramienta de dibujo a mano alzada o Bézier, agrega detalles y aplica color.
  2. Aplica el efecto de extrusión: selecciona el diseño del gatito y aplica el efecto de extrusión interactiva. Ajusta los parámetros como profundidad, ángulo y suavidad para lograr el efecto 3D deseado.
  3. Dibuja la forma de la prenda de ropa: selecciona el diseño extruido del gatito y la forma, y crea la envoltura. Ajusta los parámetros para adaptar el diseño a la prenda.
  4. Aplica la silueta interactiva: selecciona el diseño envuelto y aplica la silueta interactiva. Ajusta los parámetros para resaltar el contorno y darle un aspecto gráfico.
  5. Refina el diseño: utiliza la herramienta de selección para mover, escalar o rotar elementos. Agrega detalles adicionales como sombras, brillos o texturas.
  6. Aplica color a las diferentes partes del diseño utilizando la paleta de colores. Experimenta con combinaciones de color para lograr el estilo deseado.
  7. Guarda el proyecto en formato .cdr y exporta el diseño final en el formato adecuado (PNG, JPEG, SVG, etc.) para usarlo en proyectos de diseño de ropa.

    Herramienta seleccion Efectos Herramienta texto Herramienta silueta Herramienta trazos artísticos Efectos

PROCEDIMIENTO

  1. Crea el diseño base del banner:

    • Selecciona el texto extruido y aplica un trazo artístico desde la barra de propiedades.

    • Guarda el proyecto en formato .cdr para futuras ediciones.

Semana 9:

°09: Distorsionar y desenfocar con imágenesEfectos con imágenes Distorsionar Desenfocar





Para realizar estos diseños hize uso de las siguientes herramientas:
    Herramienta seleccion Herramienta texto Herramienta silueta Herramienta Trazos Artisticos Efectos

PROCEDIMIENTO

  1. Crea el Texto de Valores:
    • Abre CorelDRAW y crea un nuevo documento.
    • Selecciona la herramienta de texto y escribe el texto con los valores que deseas distorsionar y desenfocar.
    • Ajusta el tamaño, fuente y color del texto según tus preferencias.
  2. Convierte a Curvas:
    • Selecciona el texto y ve a Objeto > Convertir a curvas.
    • Esto convertirá el texto en objetos vectoriales que podrás editar y distorsionar.
  3. Aplica Efectos de Distorsión:
    • Selecciona el texto convertido a curvas.
    • Ve a Efectos > Distorsión > Distorsión interactiva.
    • Ajusta los parámetros de distorsión en la barra de propiedades, como el tipo de distorsión, la intensidad y la dirección, para lograr el efecto deseado.
  4. Aplica Efectos de Desenfoque:
    • Asegúrate de tener seleccionado el texto distorsionado.
    • Ve a Efectos > Desenfocar > Desenfoque Gaussiano.
    • Ajusta el radio de desenfoque en la barra de propiedades para controlar la cantidad de desenfoque aplicado al texto.
  5. Refina el Diseño:
    • Utiliza la herramienta de selección para mover, escalar o rotar el texto distorsionado y desenfocado según sea necesario.
    • Agrega detalles adicionales al diseño, como sombras, brillos o efectos de textura.
  6. Aplica Color y Estilo:
    • Selecciona las diferentes partes del texto y aplica colores utilizando la paleta de colores.
    • Experimenta con diferentes combinaciones de colores para lograr el estilo deseado.
  7. Guarda y Exporta:
    • Guarda el proyecto en formato .cdr para futuras ediciones.
    • Exporta el diseño final en el formato deseado (PNG, JPEG, SVG, etc.) para su uso en proyectos de diseño.
